# Where Should I Go?

A location-based recommendation web application that helps users discover nearby places according to their mood, available time, and budget.

The platform simplifies decision-making by combining user preferences with location data to generate personalized destination suggestions through an intuitive multi-step user experience.

---

## Overview

Choosing where to go can sometimes be more difficult than the outing itself. This project was developed to provide users with a simple and engaging way to discover suitable destinations based on their current situation and preferences.

Users can select their mood, available time, and budget, then receive recommendations tailored to their location and choices.

## System Workflow

1. User selects their current mood.
2. User specifies available time.
3. User selects a budget range.
4. The application retrieves the user's location.
5. Place data is collected through external APIs.
6. Recommendations are generated and displayed.
7. Users can explore suggested destinations.
